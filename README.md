# Akay

Akay is a small prototype for a neighborhood errand network. It is currently a front-end experience focused on exploring the core flow of creating requests, browsing errands, and chatting with others.

## Current status

This project is still in an early stage and has limited functionality. It does not yet include a backend or database, so any data shown is currently local/demo-oriented.

## What it includes

- A home screen and request flow
- A feed of errands
- A simple chat experience
- A mobile-friendly layout

## Running the project

1. Install dependencies:
   `npm install`

2. Start the development server:
   `npm run dev`
